Number Formatter
=========

A small library to round stock price according to IDX price fraction

## Installation

  `npm install idx-round`

## Usage

    var round = require('idx-round');

    var rounded = round(5019.2);
  
  
  Output should be `5025`


## Tests

  `npm test`
